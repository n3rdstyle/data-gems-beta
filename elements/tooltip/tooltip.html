Usage Examples

HTML Markup

<div class="tooltip-container"> <div class="demo-icon"> <!-- Your trigger element --> </div> <div class="tooltip tooltip--top-right">Integrated via Google</div> </div>

JavaScript Factory Function

// Create tooltip programmatically const icon = document.querySelector('.info-icon'); const tooltip = createTooltip(icon, { text: 'Integrated via Google', position: 'top-right' }); // Update text tooltip.setText('New tooltip text'); // Change position tooltip.setPosition('bottom'); // Show/hide manually tooltip.show(); tooltip.hide();

Data Attribute (Auto-init)

<div data-tooltip="Integrated via Google" data-tooltip-position="top-right"> <!-- Your trigger element --> </div> <script> // Initialize all tooltips initTooltips(); </script>
